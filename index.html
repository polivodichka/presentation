Docker

In simple terms, Docker is a software platform that simplifies the process of building, running, managing and distributing applications.

It does this by virtualizing the operating system of the computer on which it is installed and running.

EXAMPLE
1. Download Docker from https://www.docker.com/
2. Install Docker and restart your system
3. Start Docker Desktop

Docker Desktop does not start automatically after installation. To start Docker Desktop Search for Docker, and select Docker Desktop in the search results.

4. Create Dockerfile in your app folder
4. Create Dockerfile in your app folder 

						FROM node:12-alpine as build
						WORKDIR /app
						COPY package.json /app/package.json
						RUN npm install
						COPY . /app
						RUN npm run build
						
						FROM nginx:1.18-alpine
						COPY --from=build /app/build /usr/share/nginx/html
						EXPOSE 80
						CMD ["nginx", "-g", "daemon off;"]
5. Create image

docker build -t <IMAGE NAME> <PATH>


						docker build -t store .
6. Create container

docker run --rm -d --name <CONTAINER NAME> -p <PORT>:<PORT IN CONTAINER (from Dockerfile)> <IMAGE NAME>


							docker run --rm -d --name store -p 3000:80 store

Congratulations! Your first working container!

In our case we can open localhost at port 3000 and see the result: everything is works fine :)

7. Stop container

docker stop <CONTAINER NAME>


						docker stop store
Now you can share your image using Docker Hub 

						docker tag store:latest polivodichka/store:myfirstimagepush
						docker push polivodichka/store:myfirstimagepush
Then to pull image 

						docker pull polivodichka/store:myfirstimagepush
